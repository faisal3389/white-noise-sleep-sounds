Manual Xcode Steps Do in Xcode
1. Add RevenueCat SDK via SPM
  • File > Add Package Dependencies...
  • Enter URL: https://github.com/RevenueCat/purchases-ios
  • Version rule: Up to Next Major Version from 5.0.0
  • Add the RevenueCat product to the White Noise - Sleep Sounds target
2. Add Google Mobile Ads SDK via SPM
  • File > Add Package Dependencies... (again)
  • Enter URL: https://github.com/googleads/swift-package-manager-google-mobile-ads
  • Version rule: Up to Next Major Version from 12.0.0
  • Add the GoogleMobileAds product to the White Noise - Sleep Sounds target
3. Set Info.plist Path
  • Select the White Noise - Sleep Sounds target > Build Settings
  • Search for INFOPLIST_FILE
  • Set it to White Noise - Sleep Sounds/Info.plist
4. Replace RevenueCat API Key

In StoreManager.swift line 13, replace "your_key" with your actual RevenueCat public API key

5. RevenueCat Dashboard Setup
  • Create a project at app.revenuecat.com
  • Add your iOS app with bundle ID zalgo.White-Noise---Sleep-Sounds
  • Create an Entitlement called premium
  • Create a Product: com.zalgo.whitenoisesleepsounds.premium (non-consumable, $3.99)
  • Create an Offering (set as default) containing that product as a package
6. App Store Connect (for real purchases)
  • Create the IAP product com.zalgo.whitenoisesleepsounds.premium in App Store Connect
  • Link it in RevenueCat dashboard under Products
7. AdMob Setup (for real ads later)

Replace the test GADApplicationIdentifier in Info.plist with your real AdMob app ID

Replace the test ad unit ID in BannerAdView.swift with your real banner ad unit ID
